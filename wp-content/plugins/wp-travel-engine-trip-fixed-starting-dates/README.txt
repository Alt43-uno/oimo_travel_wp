Trip Fixed Starting Dates allows travel agencies and tour operators to show trip fixed starting dates and number of space available for booking.

== Description ==

This extension allows travel agencies and tour operators to show trip fixed starting dates and number of space available for booking.

For example, you may have fixed dates for a trip and only limited seats are available for the date. Then using this extension, you can added the dates and number of seats available for the dates. Booking can only be made on those dates. After a booking is made, the number of available space automatically get reduced.
